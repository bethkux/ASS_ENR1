# Features List

### Registering a Course `CORE`
As a teacher, I want to register a new course with its information so that it can be scheduled and attended by students.

### Editing a Course
As a teacher, I want to be able to edit or delete my existing courses so I can correct errors.

### Scheduling a Course `Core`
As a committee member, I need to be able to schedule an existing course to a specific time and room so I can gradually build the school schedule.

### Viewing Unscheduled Courses
As a committee member, I want to be able to see the unscheduled courses so I know which courses are missing in the schedule and how many there are.

### Viewing the Whole Schedule
As a committee member, I need to see a table with all scheduled courses (occupancy for each time and room) so I can see which slots are free.

### Collision Detection `Core`
As a committee member, I want the system to notify me when more courses are scheduled in the same room and time so I can avoid collisions in the schedule.

### Teacher Scheduling
As a teacher, I want to be able to schedule my courses so I can assign an appropriate room and time for them.

### Teacher Preference
As a teacher, I want to specify the times I am unable to teach so I will not be assigned a course at that time.

### Schedule Viewing `Core`
As a user, I want to be able to view a schedule specific to me, a room, a course, or a programme.

### Statistics Viewing `Core`
As a user, I want to be able to see the statistics from previous years, such as room utilization and teacher workload.
